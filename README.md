# Iran Stock Trade AI Artifact

Financial Machine Learning pipeline for Iranian stock market decision-support artifact.

## Objective

Develop and evaluate an AI-based trading decision-support system using historical stock market data.

## Pipeline

1. Data preparation
2. Feature engineering
3. Triple Barrier labeling audit
4. Machine learning modeling
5. Walk-forward validation
6. Economic evaluation

## Dataset

Raw stock data:
- 825 Iranian listed stocks
- Technical indicators
- Triple Barrier based labels

## Structure

- notebooks: experiments
- src: reusable code
- configs: parameters
- results: generated outputs
